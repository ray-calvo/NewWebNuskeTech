Sistema editorial para diferenciadores hospitalarios
Propósito

Este documento define las reglas editoriales obligatorias para todos los bloques de “diferenciadores hospitalarios” dentro del sitio de Nuskë Vet Center.

Su objetivo es:

mantener coherencia narrativa clínica
evitar deriva hacia copy técnico o institucional
evitar deriva hacia marketing o landing
sostener posicionamiento hospital premium
facilitar escalabilidad del contenido clínico
🎯 PRINCIPIO CENTRAL

Los diferenciadores hospitalarios no explican lo que el hospital es.

Explican:

👉 cómo cambia la decisión clínica para el paciente

Si no cambia una decisión clínica real, no es un diferenciador.

🧠 FUNCIÓN NARRATIVA

Los diferenciadores deben ayudar al usuario a entender:

cuándo importa una capacidad clínica
qué problema evita o mejora
qué cambia en el pronóstico o manejo
por qué esta capacidad existe en un hospital

No deben explicar:

la filosofía del hospital
el modelo institucional
el nivel tecnológico abstracto
conceptos clínicos generales
🧩 ESTRUCTURA OBLIGATORIA DE CADA CARD

Cada card debe responder implícitamente:

¿Qué decisión clínica cambia?
¿Cuándo importa eso?
¿Qué evita o mejora?

Si una card no puede responder esto → está mal construida.

✏️ FORMATO EDITORIAL
Título

Debe ser:

concreto
clínico
accionable
situacional
corto

Debe evitar:

abstracciones
sustantivos conceptuales
slogans
claims institucionales
Ejemplos correctos
No todo caso necesita cirugía abierta
Permite decidir sin esperar deterioro
Diagnosticar y tratar en el mismo acto
Reduce el riesgo cuando el paciente está inestable
Ejemplos incorrectos
Capacidad clínica avanzada
Menor invasión con criterio
Soporte premium especializado
Conecta evaluación y tratamiento
Subtexto

Debe:

explicar impacto clínico
ser comprensible para dueño informado
ser breve
ser situacional

Debe evitar:

meta-discurso hospitalario
lenguaje institucional
tono académico
generalidades abstractas
Ejemplo correcto

Cuando el caso lo permite, evita abrir y reduce el tiempo de recuperación.

Ejemplo incorrecto

Representa una alternativa mínimamente invasiva dentro del modelo hospitalario.

🔬 TEST EDITORIAL OBLIGATORIO

Antes de aprobar un diferenciador, validar:

¿Un dueño entiende esto en menos de 3 segundos?
¿Esto cambia una decisión clínica?
¿Esto explica impacto o solo concepto?
¿Esto podría decirlo cualquier hospital?
¿Esto suena a hospital o a brochure?

Si falla en 2 → reescribir.

⚖️ REGLA DE FRONTERAS CLÍNICAS

Cada capacidad hospitalaria debe tener diferenciadores coherentes con su rol:

Urgencias → diferenciadores operativos
Cirugía → diferenciadores de resolución
Diagnóstico → diferenciadores de precisión
Endoscopia → diferenciadores de mínima invasión

Esto evita:

solapamiento narrativo
confusión clínica
pérdida de jerarquía hospitalaria
🧭 POSICIÓN DENTRO DE LA PÁGINA

Los diferenciadores deben aparecer:

✔ después de explicar la capacidad clínica
✔ antes del CTA final

Nunca:

✖ como introducción conceptual
✖ como cierre institucional

🎯 META EDITORIAL FINAL

El usuario no debe salir pensando:

“Qué hospital tan sofisticado.”

Debe salir pensando:

“Aquí toman mejores decisiones clínicas.”

Ese es el posicionamiento hospital premium real.
