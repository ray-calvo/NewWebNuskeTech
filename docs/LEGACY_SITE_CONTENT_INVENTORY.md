# LEGACY SITE CONTENT INVENTORY

## Resumen ejecutivo

La auditoría del sitio legacy de Nuskë Vet Center muestra dos realidades distintas:

- varias páginas de servicio “normales” aportan sobre todo naming, intención y CTAs básicos
- varias landings (`/lp/*`) sí contienen módulos de contenido reutilizables de alto valor para la nueva web

Conclusión operativa:

- la nueva web no debería migrar una página por cada URL legacy sin criterio
- conviene agrupar el contenido en clusters canónicos hospitalarios
- varias URLs actuales son duplicadas o semiduplicadas y deben fusionarse
- algunas páginas comerciales subordinadas deben preservarse, pero no como núcleo del posicionamiento hospitalario

## Clusters propuestos

### Cluster 1 — Urgencias y Paciente Crítico

Rutas fuente actuales:

- `/urgencias`
- `/urgencias-24h`
- `/hospitalizacion`
- `/lp/hospitalizacion`

Intención principal:

- captación inmediata de urgencias
- confianza clínica
- explicar capacidad hospitalaria crítica

Módulos rescatables:

- hero de urgencias con CTA de llamada / WhatsApp / cómo llegar
- protocolo de emergencia (`/urgencias`)
- infraestructura médica para urgencias (`/urgencias`)
- criterios médicos de hospitalización (`/lp/hospitalizacion`)
- bloque de UCI / monitoreo / vigilancia 24h
- FAQs de urgencias
- ubicación estratégica

CTAs actuales relevantes:

- llamar ahora
- WhatsApp inmediato
- consultar disponibilidad
- abrir en Google Maps

Recomendación para nueva web:

- **página canónica**
- fusionar `urgencias`, `urgencias-24h`, `hospitalizacion` y la parte más fuerte de `lp/hospitalizacion`

- comparativa riesgo común vs protocolo Nuskë (`/lp/esterilizaciones`)
- bloque de proceso de atención ortopédica (`/ortopedia`)
- bloque pre-consulta / qué hacer antes de venir (`/ortopedia`, `/manejo-de-heridas`)
- precios de esterilización como módulo comercial subordinado, no canónico

CTAs actuales relevantes:

- agendar valoración

- WhatsApp
- llamar

Recomendación para nueva web:

- **página canónica**
- fusionar `cirugia`, `cirugia-especializada` y `lp/cirugia`
- tratar `esterilizaciones`, `manejo-de-heridas` y `ortopedia` como submódulos o futuras subpáginas de apoyo

## Módulos reutilizables detectados

### Alta prioridad de rescate

- hero clínico con CTA operativa
- protocolo de emergencia
- señales de alerta / cuándo acudir
- criterios médicos de indicación
- bloque de seguridad anestésica
- FAQ clínica
- ubicación / cómo llegar
- CTA de WhatsApp / llamada / Google Maps

### Prioridad media

- qué esperar en primera visita
- proceso de atención
- comparativas clínicas (RX vs ultrasonido, riesgo común vs protocolo Nuskë)
- aplicaciones clínicas
- checklist o preparación previa
- bloque de confianza / enfoque médico responsable

### Prioridad baja o condicionada

- precios detallados
- testimonios / reseñas de casos
- galerías de grooming
- tablas tarifarias de pensión
